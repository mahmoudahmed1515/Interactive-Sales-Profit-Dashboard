# Interactive Sales & Profit Performance Dashboard (Excel)

## Project Overview
An end-to-end data analysis project using a global retail dataset to track sales performance, profitability, and logistics efficiency.

## Key Features
- **Dynamic Dashboard:** Interactive slicers for (Region, Segment, Category, and Year).
- **Advanced Data Cleaning:** Used Power Query for ETL processes (handling nulls, duplicates, and data formatting).
- **Data Modeling:** Built a relational model using Power Pivot to connect multiple data sources.
- **DAX Measures:** Implemented custom calculations for Profit Margin, YoY growth, and Shipping Durations.
- **Visual Insights:** High-level KPIs, Top/Bottom performing products, and Shipping mode distribution.

## Tools Used
- Microsoft Excel (Advanced)
- Power Query
- Power Pivot (DAX)
- Data Visualization Principles

## 📁 How to Use
This project uses a **Data-Connection Model**. To run it on your local machine:

1. **Download both files:** Keep `Final Project.xlsx` and `Repo.xlsx` in the same folder.
2. **Open the Dashboard:** Open the `Repo.xlsx` file.
3. **Reconnect the Data Source:** - Go to the **Data** tab > **Queries & Connections**.
   - If prompted, update the file path to point to your local copy of `Final Project.xlsx`.
4. **Refresh:** Click **Data > Refresh All** to populate the charts.
5. **Interact:** Use the slicers on the right to filter insights by **Year, Region, Segment, or Category**.
